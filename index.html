CSS Box Models

Spacing Areas for Content

Name Description Dimensions
content area Contains the actual "content" such as text, images, or video. content width, content height, width, height
padding area Extends the content area to include the element's padding. Creates space around content, default centered. padding, padding-top, padding-right, padding-bottom, padding-left
border Extends the padding area to inclue the content's borders. The (in)visible outline of the content area. border, border-width
margin Extends the border area to include an empty area used to separate the element from its neighbors. margin, margin-top, margin-right, margin-bottom, margin-left
